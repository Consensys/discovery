# discovery

## Overview

This is a Java implementation of the [Discovery v5](https://github.com/ethereum/devp2p/blob/master/discv5/discv5.md)
peer discovery protocol.

## Dependency

```groovy
repositories {
    mavenCentral()
}

dependencies {
    implementation("io.consensys.protocols:discovery:<version>")
}
```
